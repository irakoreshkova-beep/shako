# Birthday Gift Site

Готовая статическая версия подарка.

## Что загружать

Загружай содержимое этой папки:

- `index.html`
- `manifest.webmanifest`
- папку `assets/`

## GitHub Pages

1. Создай новый репозиторий.
2. Загрузи в него `index.html` из этой папки.
3. В настройках репозитория открой `Pages`.
4. Выбери публикацию из ветки `main` и из корня репозитория.
5. После публикации открой ссылку GitHub Pages на телефоне и добавь сайт на экран домой.

## Supabase Static Hosting

1. Создай новый проект.
2. Открой раздел для статического сайта.
3. Загрузи `index.html`, `manifest.webmanifest` и папку `assets/`.
4. После публикации открой выданный URL на телефоне.

## Важная дата

Сейчас подарок настроен на открытие с `25 марта 2026`.

Если дату нужно поменять, в `index.html` найди строку:

`var startDate=new Date(2026,2,25);`

Месяц в JavaScript считается с нуля:

- `0` = январь
- `1` = февраль
- `2` = март
